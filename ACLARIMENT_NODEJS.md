# ℹ️ Aclariment sobre Node.js

## Node.js és GRATUÏT

Node.js és completament **gratuït i open source**. No costa diners.

- 🆓 Descàrrega gratuïta: https://nodejs.org/
- 📜 Llicència MIT (codi obert)
- 💻 Disponible per Windows, macOS i Linux

## Per què es menciona Node.js?

Node.js s'utilitza per executar el framework React (frontend modern). Però **NO és obligatori** per usar Viora.

## Opcions de Frontend per Viora

### ✅ Opció 1: Frontend Simple Python (RECOMANAT)

**No necessita Node.js**

```bash
python frontend_simple.py
```

Avantatges:
- ✅ Només Python
- ✅ Tot integrat en un servidor
- ✅ Més simple d'instal·lar
- ✅ Perfecte per començar

### ✅ Opció 2: Frontend React

**Necessita Node.js (gratuït)**

```bash
cd frontend
npm install
npm start
```

Avantatges:
- ✅ Interfície més moderna
- ✅ Millor experiència d'usuari
- ✅ Més opcions de personalització

## Recomanació

Si vols començar ràpidament: **Usa l'Opció 1** (frontend_simple.py)

Si vols una interfície més professional i no et fa res instal·lar Node.js (gratuït): **Usa l'Opció 2** (React)

## Com instal·lar Node.js (si vols)

1. Ves a https://nodejs.org/
2. Descarrega la versió LTS (recomanada)
3. Executa l'instal·lador
4. Verifica: `node --version`

**Cost: 0€** - És totalment gratuït!
